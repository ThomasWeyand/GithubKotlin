# Github Kotlin
Aplicativo para consultar a API do Github e trazer os repositórios com mais estrelas em Kotlin.

## Características da aplicação
  * Kotlin
  * Paging 3 library
  * MVVM
  * Clean architecture
  * Scroll infinito
  * Coroutines
  * Kotlin Flow
  * Koin para injeção de dependência
  * View binding
  * Retrofit
